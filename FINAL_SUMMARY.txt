1. NODE.JS DEPENDENCIES INSTALLIEREN
   npm install
   
   Das installiert:
   - express (Web Server)
   - stripe (Payment Processing)
   - axios (HTTP Requests)
   - cors (Cross-Origin Requests)
   - dotenv (Environment Variables)

2. PAYPAL SETUP (Optional)
   1. Gehe zu: https://developer.paypal.com/dashboard/
   2. Apps & Credentials → Sandbox
   3. Client ID & Secret kopieren
   4. In .env eintragen:
      PAYPAL_CLIENT_ID=...
      PAYPAL_CLIENT_SECRET=...

3. STRIPE SETUP (Optional)
   1. Gehe zu: https://stripe.com/
   2. Developers → API Keys (Test Mode)
   3. Secret Key kopieren (sk_test_...)
   4. In .env eintragen:
      STRIPE_SECRET_KEY=sk_test_...

4. SERVER STARTEN
   npm start
   
   Server läuft dann auf: http://127.0.0.1:5000
